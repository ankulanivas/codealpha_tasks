# 📁 File Organizer | CodeAlpha Python Internship

## 📌 Project Overview

This project automates file organization using Python. It scans a source folder, identifies image files (`.jpg`), and moves them into a destination folder automatically.

This project was developed as part of the **CodeAlpha Python Programming Internship**.

---

## 🚀 Features

* Moves all `.jpg` files automatically.
* Creates the destination folder if it doesn't exist.
* Displays a success message for each moved file.
* Ignores unsupported file types.
* Simple and beginner-friendly automation project.

---

## 🛠 Technologies Used

* Python 3
* `os` Module
* `shutil` Module

---

## 📚 Python Concepts Used

* File Handling
* Loops
* Conditional Statements
* Modules (`os`, `shutil`)
* String Methods
* Directory Operations

## ▶️ How to Run

1. Place image files inside the source folder.
2. Run:

```bash
python organizer.py
```

3. All `.jpg` files will be moved automatically.
