# 📈 Stock Portfolio Tracker | Python Programming

## 📌 Project Overview

This project calculates the total investment value of a stock portfolio using predefined stock prices and user-entered quantities.

The result is displayed on the console and also saved to a text file.

Developed as part of the **Python Programming Learning**.

---

## 🚀 Features

* User enters stock name and quantity.
* Uses predefined stock prices.
* Calculates total investment value.
* Saves the result to a text file.
* Simple dictionary-based implementation.

---

## 🛠 Technologies Used

* Python 3

---

## 📚 Python Concepts Used

* Dictionaries
* User Input
* Arithmetic Operations
* File Handling
* String Methods

## ▶️ How to Run

Run:

```bash
python portfolio.py
```

Enter the stock name and quantity when prompted.

The total investment value will be displayed and saved in `total.txt`.
