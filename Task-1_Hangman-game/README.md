# 🎮 Hangman Game |Python Programming

## 📌 Project Overview

This is a text-based Hangman game developed using Python. The computer randomly selects a word, and the player must guess it one letter at a time before running out of lives.

This project was created as part of  ** Python Programming learning **.

---

## 🚀 Features

* Random word selection.
* Six incorrect guesses allowed.
* Displays guessed letters and remaining blanks.
* Tracks remaining lives.
* Win and lose conditions.
* Console-based gameplay.

---

## 🛠 Technologies Used

* Python 3
* `random` Module

---

## 📚 Python Concepts Used

* `while` Loop
* `for` Loop
* `enumerate()`
* Conditional Statements

## ▶️ How to Run

Run the following command:

```bash
python hangman.py
```

Guess one letter at a time until you either guess the complete word or lose all six lives.

---

✨ User Experience

To make the console output more readable and user-friendly, the game uses simple visual symbols:

✅ Indicates a correct guess.
❌ Indicates an incorrect guess.
🎉 Indicates the player has successfully guessed the word.
💀 Indicates the game is over after all lives are exhausted.

These symbols improve readability and make the console-based gameplay more engaging.
